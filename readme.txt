El proyecto tiene por objetivo desarrollar un sistema de base de datos 
para un proyecto de empresa de construccion.
Se controlará el inventario de materiales de proyecto.
Las funcionalidades son:
1. Ingresar materiales (Orden de compra, nombre, cantidad, precio)
2. Mostrar materiales
3. Listar materiales
4. Retirar materiales
5. Salir




- Registro de users (user, contraseña)
- Registro de proveedores (ID, RUC, nombre, telefono, email)
- Registro de personas con acceso a ingreso y retiro de materiales (IDpersona, nombre, cargo)
- Registro de materiales (ID,  orden de compra, producto, cantidad, precio)
- Registro de movimientos (ID, IDresponsable, ID producto, cantidad, tipomovimiento)
